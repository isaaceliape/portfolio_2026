Overview

mark-board is an innovative Kanban board application that stores all your tasks in plain markdown files. Unlike traditional project management tools, mark-board keeps your data human-readable and version-control friendly while providing a beautiful, modern UI for managing workflows.

Key Features

  • Markdown-based: All tasks and boards stored as readable markdown files
  • AI-powered: Get workflow suggestions and optimization recommendations powered by AI
  • Git-friendly: Version control your project boards with git
  • Drag & Drop: Intuitive card management across columns
  • Custom Columns: Create workflows that match your team's process
  • Real-time Sync: Automatic file synchronization as you work

Technical Implementation

Built with vanilla JavaScript and a focus on file-based data storage. The application bridges the gap between developer-friendly markdown and user-friendly UI, using clever parsing and serialization to maintain data integrity while providing real-time updates.

Use Cases

Perfect for developers and teams who prefer keeping project data in git repositories, writers managing editorial workflows, or anyone who wants their task management data to be portable and human-readable. The AI integration helps optimize workflows and suggest improvements based on task patterns.
